SauceLabDemo :

This project demonstrates an automation testing framework built using Java, Selenium, Maven, and TestNG, developed in IntelliJ IDEA.

## Project Structure :
src/main/jav
- pages - All POM classes (Object Repository)
- util - Reusable utilities, Listeners
- base - Base class for WebDriver setup
src/test/java
- tests -Test classes
- resources/testData -Test data / Properties files

- testng.xml -TestNG Suite file
- pom.xml   - Maven dependencies
- allure-results - Auto generated by Allure
- allure-report - Final generated report

-------------------------------------------------------
## Setup
1. JDK 21 must be installed locally, and the required environment variables should be configured.
2. Maven must also be installed and configured.

## How to Run
1. Install Java and Maven.
2. Run tests: testng.xml -- file
